# iMap
>生成专属于你自己的旅行地图

##项目动机
一直想做一个属于自己的旅行地图，奈何市面上都不太好用，抱着学习Echarts和electron的初衷，就决定自己动手撸一个

##技术栈
* Vue.js
* Vuex
* Vue-router
* axios
* lodash
* electron
* leancloud（express）
* Echarts


##iMap功能
已完成

* [x]  导入Excel表格生成地图
* [x] 手动输入地点数据生成地图
* [x] 自动保存
* [x]  自动更新

* [x] 切换中国与世界地图

## 项目启动
```
# 安装依赖
yarn

# 启动项目
yarn run dev

# 项目打包（darwin、 mas、 linux 和 win32）
npm run build

```
